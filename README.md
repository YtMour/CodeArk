## 🚢 代码方舟 (CodeArk)

**企业级私有代码资产封存与灾难恢复解决方案**

`CodeArk` 是一款面向开发者的分布式备份系统，旨在建立除 GitHub 之外的第二道“数字防线”。它通过本地客户端加密压缩与远程存储解耦的架构，确保在本地开发环境（如虚拟机、物理机）发生灾难性故障时，能够通过“方舟”实现环境级的快速还原。

### 1. 业务价值 (Value Proposition)

- **隐私隔离：** 解决 `.env`、私钥、本地数据库等无法上传至公网仓库的备份痛点。
- **环境对齐：** 备份不仅包含代码，还可包含编译产物与配置快照，确保恢复后无需重新配置环境。
- **安全合规：** 采用端到端加密（E2EE），存储端仅可见 AES 加密后的二进制流。

### 2. 核心功能模块

- **多模式触发引擎：** 支持手动触发、Cron 定时触发、以及基于文件系统监听的“静默期”触发。
- **双层加密流水线：** 强制性 ZIP 密码层 + 可选性 AES-256-GCM 载荷加密层。
- **多源存储适配器：** 原生支持 S3 协议、SMB/CIFS、SFTP 及本地磁盘挂载。
- **版本演进管理：** 记录备份元数据，支持按时间轴进行增量查看与单文件/全量恢复。

### 3. 快速部署 (Quick Start)

项目采用 Docker Compose 进行一键化中枢部署：

YAML

```
services:
  codeark-server:
    image: codeark/backend:v1.0
    restart: always
    environment:
      - AR_AUTH_TOKEN=${MASTER_TOKEN}
      - DB_DSN=postgres://user:pass@db:5432/ark
    ports:
      - "9000:9000"
  codeark-ui:
    image: codeark/frontend:v1.0
    ports:
      - "80:80"
```
