## 🚢 代码方舟 | CodeArk

> **Secure. Private. Irreplaceable.** > **当环境崩塌，方舟即是你的数字原点。**

`CodeArk` 是一款专注于**极端环境恢复**与**私有资产封存**的开发者工作站。它通过分布式架构，将你散落在各处的本地工程、配置文件及开发环境，经过双重工业级加密后，安全地引渡至私有云或局域网存储中。

### 1. 核心价值主张

- **非侵入式并行：** 与 GitHub 等版本控制系统并行，侧重于“环境全量快照”而非“代码增量更新”。
- **灾难恢复确定性：** 针对虚拟机损坏、硬盘故障、误删等场景，提供一键式、带环境参数的还原能力。
- **主权归属：** 所有的加密密钥与存储路径均由用户私有化部署，不经过任何第三方中转。

### 2. 系统组成

| **组件**       | **名称**     | **核心技术**        | **描述**                                                |
| -------------- | ------------ | ------------------- | ------------------------------------------------------- |
| **中枢控制塔** | `Ark-Server` | Go / Vue 3 / Docker | 管理端。负责策略分发、存储对接及全局任务监控。          |
| **作业采集器** | `Ark-Client` | Go / Wails (Native) | 客户端。负责本地文件监听、ZIP+AES 双重加密及流式上传。  |
| **能源储备仓** | `Storage`    | S3 / WebDAV / Local | 存储层。支持各种 OSS 协议、局域网 SMB 及 VPS 物理路径。 |
